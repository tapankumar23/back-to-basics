// ─── KMP: Build Failure Function ─────────────────────────────────
int[] buildFailure(String p) {
    int m = p.length();
    int[] f = new int[m];
    int k = 0;
    for (int i = 1; i < m; i++) {
        while (k > 0 && p.charAt(k) != p.charAt(i)) k = f[k - 1];
        if (p.charAt(k) == p.charAt(i)) k++;
        f[i] = k;
    }
    return f;
}

// ─── KMP: Search — O(n + m), no text backtracking ────────────────
List<Integer> kmpSearch(String t, String p) {
    List<Integer> res = new ArrayList<>();
    int[] f = buildFailure(p);
    int k = 0;
    for (int i = 0; i < t.length(); i++) {
        while (k > 0 && p.charAt(k) != t.charAt(i)) k = f[k - 1];
        if (p.charAt(k) == t.charAt(i)) k++;
        if (k == p.length()) {
            res.add(i - k + 1);
            k = f[k - 1];   // continue searching for overlapping matches
        }
    }
    return res;
}

// ─── Basic Calculator I (handles +, -, parentheses) ──────────────
public int calculate(String s) {
    ArrayDeque<Integer> stack = new ArrayDeque<>();
    int result = 0, num = 0, sign = 1;
    for (char c : s.toCharArray()) {
        if (Character.isDigit(c)) num = num * 10 + (c - '0');
        else if (c == '+') { result += sign * num; num = 0; sign = 1; }
        else if (c == '-') { result += sign * num; num = 0; sign = -1; }
        else if (c == '(') { stack.push(result); stack.push(sign); result = 0; sign = 1; }
        else if (c == ')') {
            result += sign * num; num = 0;
            result *= stack.pop();   // sign before '('
            result += stack.pop();   // result before '('
        }
    }
    return result + sign * num;
}

Staff-Level Angles

"KMP on 1 TB log files — how to parallelize?"
Chunk with overlap: split the text into chunks; each worker receives an extra pattern_length − 1 bytes at its right boundary so cross-boundary matches are not missed.

Independent KMP per chunk: each worker runs a full KMP search on its slice in parallel; boundary matches get stitched by the coordinator.

SIMD / GPU acceleration: vectorized character comparison with AVX-512 can process 64 bytes per cycle; GPU parallelism useful when the pattern set is large (multi-pattern Aho-Corasick on GPU shared memory).

Streaming: for live logs, maintain a small sliding buffer at shard boundaries in Flink / Kafka Streams — emit only fully verified matches downstream.
"Spreadsheet formula evaluator with 10 M cell references — parallelize dependency resolution?"
Build DAG first: parse every formula into an expression tree (two-stack approach) and emit directed edges dependency → dependent. Cycle detection = circular reference error.

Topological sort: Kahn's algorithm produces evaluation layers. Cells with in-degree 0 form layer 0 and can be evaluated immediately in parallel.

Parallel layer execution: each layer is a data-parallel batch — dispatch to a thread pool or GPU kernel. Write results to a shared value store before advancing to the next layer.

Cache-aware chunking: group cells by sheet or contiguous region to maximize L1/L2 cache reuse during formula evaluation; avoid cross-NUMA random reads for large value arrays.
