Kahn's BFS & DFS Postorder

// Kahn's Algorithm — BFS in-degree approach
int[] kahn(int n, int[][] prereqs) {
    List<List<Integer>> adj = new ArrayList<>();
    int[] ind = new int[n];
    for (int i = 0; i < n; i++) adj.add(new ArrayList<>());
    for (int[] p : prereqs) {
        adj.get(p[1]).add(p[0]);   // p[1] -> p[0]
        ind[p[0]]++;
    }
    Queue<Integer> q = new ArrayDeque<>();
    for (int i = 0; i < n; i++) if (ind[i] == 0) q.offer(i);
    int[] ord = new int[n];
    int idx = 0;
    while (!q.isEmpty()) {
        int u = q.poll();
        ord[idx++] = u;
        for (int v : adj.get(u))
            if (--ind[v] == 0) q.offer(v);
    }
    return idx == n ? ord : new int[0]; // empty = cycle detected
}

// DFS Postorder — push to stack after all neighbours visited
void dfsTopo(int u, List<List<Integer>> adj,
             boolean[] vis, ArrayDeque<Integer> res) {
    vis[u] = true;
    for (int v : adj.get(u))
        if (!vis[v]) dfsTopo(v, adj, vis, res);
    res.push(u); // push AFTER all descendants
}

// Cycle detection in DFS (three-color marking)
boolean hasCycle(int u, List<List<Integer>> adj,
                 int[] color) { // 0=white,1=gray,2=black
    color[u] = 1;
    for (int v : adj.get(u)) {
        if (color[v] == 1) return true;  // back-edge = cycle
        if (color[v] == 0 && hasCycle(v, adj, color)) return true;
    }
    color[u] = 2;
    return false;
}

Staff Engineer Angles

Production Scale Questions

"100k-module build graph — detect cycles incrementally as developers add deps?"
Incremental cycle detection: maintain a timestamp-based DFS forest. On each new edge u->v, do a targeted DFS from v back to u bounded by v's subtree. If u is reachable, reject the edge. Amortised cost O(V+E) total. For tighter guarantees, use Tarjan's SCC on the full graph periodically and cache SCCs of size >1. In monorepo CI: shadow-graph the edge before writing it; fail the PR if it closes a cycle.
"Schedule 10k microservice deployments with deps across 50 regions — maximize parallelism?"
Critical path analysis on a DAG: assign each node a level = longest path from any source. Services at the same level have no mutual dependency and can deploy in parallel. Kahn's BFS naturally produces these levels — process the queue in waves. Across 50 regions, partition the DAG into regional sub-DAGs sharing only cross-region edges (model those as latency-weighted edges). The answer is O(longest chain length) deployment rounds.
